# [!UICONTROL Google Performance Max Placement Report]

*Gäller endast [!DNL Google Ads]-konton med maximala prestandakampanjer*

[!UICONTROL Google Performance Max Placement Report] innehåller visningsdata för placeringar i [!DNL Google Ads] maximala prestandakampanjer inom ett angivet konto. Rapporten innehåller avtryck för varje inkluderad placering efter mål-URL. Som standard innehåller data en rad för varje kombination av placerings- och målwebbadress som fick minst ett intryck i det angivna dataområdet, även om placeringen togs bort under rapporteringsperioden. Raderna är i stigande ordning som standard i kampanjer.

Använd den här rapporten för att hitta URL:er som ska exkluderas och för att förbättra prestandan för max-kampanjer.
