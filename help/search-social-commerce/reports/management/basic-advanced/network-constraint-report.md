# [!UICONTROL Network Constraint Report]

[!UICONTROL Network Constraint Report] visar alla angivna minimi- och maximibudgetar för annonsering samt den faktiska kostnaden. Rapporten innehåller uppgifter om kostnader, klick, intrycket, genomgång och (valfritt) konvertering för en eller flera portföljer. Som standard innehåller data en rad för varje tillämpligt annonsnätverkskonto i det angivna datumintervallet, och raderna är i stigande ordning först efter portfölj och sedan efter annonsnätverk.

Data är tillgängliga från och med den 15 oktober 2022.<!-- [Later: You can view data for the previous NN days.] -->
