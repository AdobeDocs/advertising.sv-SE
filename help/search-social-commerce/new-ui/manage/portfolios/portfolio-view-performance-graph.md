# (Nytt användargränssnitt) Visa ett prestandagram i vyn [!UICONTROL Portfolios]

*Beta-funktion*

Du kan öppna och konfigurera ett prestandagram med upp till tre mätvärden som summeras för alla portföljer i vyn för det angivna datumintervallet.

## Visa ett prestandadiagram

1. Ovanför datatabellen klickar du på ![Diagram](/help/search-social-commerce/assets/charts.png "Diagram").

1. (Valfritt) Ange valuta och upp till tre mätvärden som ska ingå i diagrammet.

## Dölja ett synligt prestandadiagram

* Ovanför datatabellen klickar du på ![Diagram](/help/search-social-commerce/assets/charts.png "Diagram").
