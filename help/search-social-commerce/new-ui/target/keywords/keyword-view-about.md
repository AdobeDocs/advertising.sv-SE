# (Nytt användargränssnitt) Om vyn [!UICONTROL Keywords]

*Beta-funktion*

Vyn [!UICONTROL Manage] > [!UICONTROL Keywords] innehåller två flikar:

* På fliken [!UICONTROL Keywords] visas alla nyckelord i den filtrerade vyn för det valda annonserarkontot.

* Fliken [!UICONTROL Negatives] visar alla negativa nyckelord i den filtrerade vyn för det valda annonserarkontot.
