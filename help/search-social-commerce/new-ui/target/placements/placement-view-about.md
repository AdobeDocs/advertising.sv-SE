# (Nytt användargränssnitt) Om vyn [!UICONTROL Placements]

*Beta-funktion*

Vyn [!UICONTROL Manage] > [!UICONTROL Placements] innehåller två flikar:

* På fliken [!UICONTROL Placements] visas alla placeringar i den filtrerade vyn för det valda annonserarkontot.

* Fliken [!UICONTROL Negatives] visar alla negativa placeringar i den filtrerade vyn för det valda annonserarkontot.
