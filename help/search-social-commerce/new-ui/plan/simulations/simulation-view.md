# Visa simuleringsinformation

*Beta-funktion*

Du kan visa simuleringsdetaljer på skärmen som ett diagram eller en tabell.

1. Klicka på **[!UICONTROL Plan]>[!UICONTROL Simulations]** på huvudmenyn.

1. Klicka på simuleringsnamnet.

   Simuleringen öppnas i diagramvyn.

1. (Valfritt) Gör något av följande i diagramvyn:

   * Om du vill ändra måttet för vilket beräknade data visas för varje utgiftsmålnivå (steg) väljer du ett nytt mått på menyn **[!UICONTROL Y-Axis Metric]**.

   * (Endast veckosimuleringar) Ange ett nytt **[!UICONTROL Target Mid Point]**-värde om du vill se de förväntade data som använder det värdet.

   * (Endast veckosimuleringar) Välj om du vill visa data **[!UICONTROL Constraints]**, *[!UICONTROL With Constraints]* eller *[!UICONTROL Without Constraints]* på menyn *[!UICONTROL With and Without]*.

1. (Valfritt) Om du vill öppna tabellvyn klickar du på fliken **[!UICONTROL Table View]**.

1. (Valfritt) Om du vill visa de skrivskyddade inställningarna för den tillämpliga portföljen klickar du på **[!UICONTROL Portfolio Settings]** i det övre högra hörnet.
