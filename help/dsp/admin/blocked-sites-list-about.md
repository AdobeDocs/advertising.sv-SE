# Om blockerade webbplatser på kontonivå och annonsörnivå

Du kan redigera listan över blockerade webbplatser som används för hela DSP-kontot och ytterligare listor för enskilda annonsörer i kontot.

Blockerade platser listar definierar uppsättningar av mål som ska uteslutas för dina placeringar. Varje lista kan bestå av webbplatsdomäner på den översta nivån och underdomäner på alla nivåer (till exempel example.com, my.example.com eller my.new.example.com) och mobilapp-ID:n eller paketnamn (till exempel com.example.app123).

Annonsnivålistor åsidosätter kontonivålistor.

>[!NOTE]
>
>* Listor med blockerade webbplatser på kontonivå och annonsörnivå används utöver den globala listan [Blockerade webbplatser](/help/dsp/introduction/features/brand-safety-media-quality.md#global-blocked-sites) i DSP, som innehåller webbplatser som inte betraktas som säkra för annonser.
>* Användarna kan också lägga till riktade webbplatser på alla platser.
>* Listor med blockerade webbplatser åsidosätter alltid målwebbplatslistor. Om en placering båda utesluter och innehåller samma mål för en annons, utesluts målet.
