# Få hjälp

Du kan skicka en AI-assisterad fråga till alla [!DNL DSP]- och (annonsörer med Advertising Creative) [!DNL Creative] -dokumentation eller öppna en fullständig användarhandbok för [!DNL DSP]. Kontakta oss om dokumentationen inte svarar på dina frågor.

## Sök efter information med hjälp av ett AI-assisterat chattgränssnitt

Se &quot;[Sök efter produktdokumentation med hjälp av ett AI-assisterat chattgränssnitt](/help/dsp/agent-chat.md).&quot;

## Öppna en fullständig användarhandbok

Om du vill öppna en fullständig användarhandbok som förklarar hur du använder alla funktioner klickar du på **[!UICONTROL Help]** på huvudmenyn.

## Kontakt [!DNL Adobe]

Gör följande för produkt- eller kontoproblem:

* (Annonsörer med självbetjäningsavtal) Öppna en biljett för din organisation på [https://experienceleague.adobe.com/home?lang=sv-SE#support](https://experienceleague.adobe.com/home?lang=sv-SE&support-tab=home#support).

  Välj [!UICONTROL Advertising - DSP] för produkten.

* (Annonsörer med fullservicekontrakt) Kontakta Adobe Account Team.

Ange detaljerade steg för att återskapa problemet, relevanta data, webbläsarens namn och version samt information om den preliminära felsökningen som du har utfört i biljetten eller i kommunikationen. Ta inte bort rapporten för rapportproblem och ange rapportnamnet och URL-adressen.
